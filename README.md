# Credit Card Fraud Detection

## Objective

Build a machine learning model to detect fraudulent credit card transactions.

## Technologies Used

* Python
* Pandas
* Scikit-Learn

## Algorithm Used

* Logistic Regression

## Steps Performed

1. Loaded the credit card transaction dataset.
2. Selected a sample of the dataset for faster training.
3. Split the data into training and testing sets.
4. Trained a Logistic Regression model.
5. Evaluated model performance using Accuracy Score and Classification Report.
6. Predicted whether a sample transaction was fraudulent or genuine.

## Result

The model achieved approximately 99.95% accuracy and successfully classified transactions as fraudulent or genuine.
